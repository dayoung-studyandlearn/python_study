# 데이터 가공하기

## 1. 데이터 전처리_ 원하는 형태로 데이터 가공하기

데이터전처리 : 분석에 적합하게 가공하는 작업


일부를 추출하거나 종류별로 나누거나, 여러데이터를 합치는 등의 가공이 필요할 때에 작업을 거치는 것을 전처리라고 한다.

- pandas : 데이터 전처리 작업에 가장 많이 사용되는 패키지
  |함수|기능|
  |---|---|
  |query()|행 추출|
  |df[]|열(변수) 추출|
  |sort_values()|정렬|
  |groupby()|집단별로 나누기|
  |assign()|변수 추가|
  |agg()|통계치 구하기|
  |merge()|데이터 합치기(열)|
  |concat()|데이터 합치기(행)|
